# Smart Green House Monitoring System

## Description
This project is a C# based Arduino program designed to monitor and control various parameters of a smart greenhouse such as temperature, humidity, water level, and soil moisture.

## Features
- Real-time monitoring of environmental conditions
- Automated alerts for critical conditions (e.g., low water level, high temperature)
- LCD display for visual feedback

## Installation
1. Connect the Arduino board to your computer.
2. Install the necessary libraries (`LiquidCrystal`, `DHT`) using the Arduino IDE library manager.
3. Upload the code to your Arduino board.

## Usage
Upload the code to your Arduino board and observe the LCD display for real-time environmental data and alerts.

## Tech Stack
- C#
- Arduino
- LiquidCrystal Library
- DHT Sensor
